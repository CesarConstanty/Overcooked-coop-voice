Tutoriel - Condition EA (Explicabilité Auditive)

Bienvenue dans ce bloc d'expérimentation !

Dans cette condition, vous allez collaborer avec un agent intelligent qui vous aidera à cuisiner. La communication des intentions se fera exclusivement par des messages vocaux. Vous devez écouter attentivement ces annonces pour coordonner vos actions.

Particularités de cette condition :
  • Sons d'ingrédients : Chaque ingrédient sera annoncé par une voix distincte.
  • Sons de recette : Chaque recette à préparer sera également annoncée.
  • Communication auditive : Votre performance dépendra de votre attention aux signaux sonores.
