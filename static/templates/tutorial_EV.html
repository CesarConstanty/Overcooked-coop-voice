Tutoriel - Condition EV (Explicabilité Visuelle)

Bienvenue dans ce bloc d'expérimentation !

Dans cette condition, vous allez collaborer avec un agent intelligent qui vous aidera à cuisiner. L'agent vous communiquera ses intentions et ses actions grâce à des indices visuels.

Particularités de cette condition :
  • Interface enrichie : Des informations visuelles supplémentaires seront affichées à l'écran
  • Indicateurs de recette : Vous verrez des indices visuels sur les recettes à préparer
  • Indicateurs d'ingrédients : Des signaux visuels vous informeront sur la manipulation des ingrédients par l'agent
  • Interface augmentée : L'écran affichera plus d'informations pour faciliter la coordination

Objectif : Utilisez les informations visuelles supplémentaires pour mieux collaborer avec l'agent et maximiser votre efficacité.

Note : Prêtez attention aux nouveaux éléments visuels qui apparaîtront pendant le jeu pour optimiser votre coordination.
