Tutorial : Visual communication

Visual communication of intentions

In this experimental condition, the artificial agent communicates its intentions using visual bubbles above its head. These bubbles clarify which recipe the agent is pursuing and which asset it is currently seeking in the game.

Sequence explanation

The intentions will be communicated in the following order:

Recipe
→
Asset
→
Following asset
→
Next recipe
