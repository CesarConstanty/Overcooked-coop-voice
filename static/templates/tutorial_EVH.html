Tutoriel - Condition EVH

Communication visuelle des intentions

Dans cette condition expérimentale, l'IA communique ses intentions via des bulles visuelles qui apparaissent au-dessus de sa tête.

Séquence d'affichage

Les intentions sont communiquées dans l'ordre suivant :

Recette
→
Asset
→
Asset +1
→
Recette +1

Types d'intentions

  • Recette : L'image de la recette que l'IA souhaite préparer (bordure bleue)
  • Assets : Les objets que l'IA cherche à atteindre (O=Oignon, T=Tomate, D=Assiette, S=Service) (bordure verte)
  • Recette suivante : La prochaine recette prévue (bordure grise)

Durées d'affichage

Chaque type d'intention a une durée d'affichage différente :

  • Recette : 2 secondes
  • Asset : 1.5 secondes
  • Recette suivante : 1 seconde

Points importants

  • Les bulles suivent automatiquement le déplacement de l'IA
  • Elles apparaissent directement au-dessus de la tête du personnage
  • La séquence se répète quand les intentions changent
  • Seuls les assets O, T, D, S sont communiqués visuellement
